## Overview

PromptPal is a stdlib-only Python 3.10+ CLI that pipes a raw prompt through a system prompt and returns an improved version, optionally entering an interactive accept/iterate/discard refinement loop. The stack is pure Python (no third-party runtime dependencies), bash for launchers and installers, and pytest (via `uv run`) for testing. The single entry point is `bin/promptpal` (bash) which guards against a misrouted WSL HOME then execs `promptpal_main.py` by absolute path, delegating to `core/cli.py` вЂ” the sole orchestration layer owning all I/O, flags, and exit codes. Below `cli.py` sit pure-functional modules: two backends (subprocess `claude` CLI via NDJSON stream-json, or Anthropic HTTP API via SSE), plus config, history, diff, loop, platform, and atomic I/O utilities, all with injectable seams for testing. The installed command runs from a managed snapshot copied into `~/.promptpal/lib/` by `install.sh`; a winget package ships a thin PowerShell/CMD launcher that forwards into WSL.
